# Product Overview

塔罗分析 (Tarot Analysis) is a digital tarot card divination system built with React and TypeScript. The application provides users with an interactive tarot reading experience featuring:

- Complete 78-card tarot deck database (22 Major Arcana + 56 Minor Arcana)
- Multiple classic spread layouts (single card, three-card, Celtic cross)
- Intelligent card interpretation and analysis
- Responsive design for mobile and desktop
- Local storage for reading history
- Elegant tarot-themed UI with Chinese language support

The system allows users to perform online tarot card draws, receive detailed interpretations, and maintain a personal history of their readings for reflection and guidance.
